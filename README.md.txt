# Student Productivity Tracker

## Author

Bhumika Sharma



About the Project

This is a  Python project made to help students keep track of their daily tasks. Sometimes it becomes difficult to manage studies and assignments together, so this program helps in organizing tasks.


Problem

Students often forget tasks or are unable to manage time properly, which affects productivity.



 Solution

This project allows users to:

* Add tasks
* View tasks
* Mark tasks as completed
* Track progress


 How to Run

1. Make sure Python is installed
2. Open the folder in terminal
3. Run:

python main.py


 What I Learned

I learned how to use lists, dictionaries, loops, and functions in Python and how to build a simple real-life project.
